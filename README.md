# hayate.js  
Next generation web Infrastructure.

## Prerequisites

- [gcc4.7 or later][libv8][libevent2]

## Feature
* High performance
	- Support multithread enviroment.

* simplicity
 	- Internal Web server
  	- Support JavaScript

## Support
	OSX 10.9
	Linux(Not Testing)
	Windows(T.B.D)

## Usage
	Run app
	$./hayate 8080 4 web.js
	you can connect Server's IP using http protocol.
